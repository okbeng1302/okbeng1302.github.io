二分法查找原始版-查找某个元素的下标(任意一个)

复杂度：总共有N个元素，每次查找的区间大小就是N,N/2，N/4…N/2^k k是循环的次数
由于N/2^k 取整后 >= 1,也就是 N/2^k = 1; k = log2n
时间复杂度是: O(logn)
